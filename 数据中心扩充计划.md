### 🟠 P1：前复权日K线数据

**方案A：直接下载存储**
- 优点：数据独立，访问快速
- 缺点：占用空间翻倍（~3.2GB）

**方案B：实时转换（推荐）**
- 优点：节省空间，灵活性高
- 缺点：需要转换逻辑

**推荐实现：**
```python
# 在数据加载模块添加转换函数
def get_stock_daily_qfq(ts_code: str) -> pd.DataFrame:
    """从后复权数据转换为前复权"""
    hfq_df = pd.read_parquet(f'{DATA_CENTER}/stock/daily_hfq/{ts_code}.parquet')
    
    # 获取最新价格作为基准
    latest_price = hfq_df.iloc[-1]['close']
    
    # 计算复权因子
    adj_factor = latest_price / hfq_df['close']
    
    # 转换价格
    qfq_df = hfq_df.copy()
    qfq_df['close'] = hfq_df['close'] * adj_factor
    qfq_df['open'] = hfq_df['open'] * adj_factor
    qfq_df['high'] = hfq_df['high'] * adj_factor
    qfq_df['low'] = hfq_df['low'] * adj_factor
    
    return qfq_df
```
