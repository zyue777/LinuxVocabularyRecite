## 📊 数据表详细说明

> **🆕 新增数据表**（2025-11-19更新）:
> - **市场数据** (market/): 融资融券交易汇总、融资融券交易明细、沪深港通资金流向
> - **资金流向数据** (stock/moneyflow/): 个股资金流向，包含小单、中单、大单、特大单
> - **前复权数据** (stock/daily_qfq/): 股票日K线前复权数据，自动从后复权转换并存储
> 
> **📝 说明**:
> - **市场元数据** (market_metadata/): 此目录为可选功能，可通过 `data_utils.generate_market_metadata()` 生成，用于快速筛选创业板、科创板等不同市场的股票
> - **策略信号数据** (signals/): Alpha策略历史信号数据，用于策略回测和分析

**⚠️ 重要说明**:
- 前复权数据与后复权数据格式完全一致，只是价格字段已进行前复权调整
- 前复权数据在最新日期与后复权数据价格一致
- 历史价格按比例调整，便于计算历史收益率

#### 使用示例

```python
import pandas as pd

# 按日期筛选
df_2024 = df_qfq[df_qfq['trade_date'] >= '20240101']

# 计算收益率（前复权数据更适合计算历史收益率）
df_qfq['return'] = df_qfq['close'].pct_change()

# 批量读取多只股票
stock_codes = ['000001.SZ', '000002.SZ', '600519.SH']
qfq_data = {}
for code in stock_codes:
    file_path = f'{DATA_CENTER}/stock/daily_qfq/{code}.parquet'
    if os.path.exists(file_path):
        qfq_data[code] = pd.read_parquet(file_path)
```

#### 字段说明

| 字段名             | 数据类型   | 说明      | 单位       | 用途                    |

| `circ_mv`       | float  | 流通市值    | 万元       | 筛选指标                  |

#### 使用示例

```python
import pandas as pd

# 数据中心路径
DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'

# 读取所有股票的每日基础指标
df = pd.read_parquet(f'{DATA_CENTER}/stock/daily_basic/daily_basic_all.parquet')

# 筛选某一天的数据
df_20240101 = df[df['trade_date'] == '20240101']

# 按市值排序
df_sorted = df_20240101.sort_values('total_mv', ascending=False)

# 计算账面市值比（B/M）用于HML因子
df['bm_ratio'] = 1 / df['pb']

# 市值分组（用于SMB因子）
df['size_group'] = pd.qcut(df['total_mv'], q=2, labels=['Small', 'Big'])

# 价值分组（用于HML因子）
df['value_group'] = pd.qcut(df['bm_ratio'], q=3, labels=['Low', 'Medium', 'High'])
```

**资金流向分类标准**:
- **小单**: 单笔成交金额 < 4万元
- **中单**: 单笔成交金额 4-20万元
- **大单**: 单笔成交金额 20-100万元
- **特大单**: 单笔成交金额 > 100万元

#### 使用示例

```python
import pandas as pd

# 数据中心路径
DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'

# 读取单只股票资金流向数据
df_mf = pd.read_parquet(f'{DATA_CENTER}/stock/moneyflow/000001.SZ.parquet')

# 计算散户净流入（小单净流入）
df_mf['retail_net_inflow'] = df_mf['buy_sm_amount'] - df_mf['sell_sm_amount']

# 计算机构净流入（大单+特大单净流入）
df_mf['institution_net_inflow'] = (
    (df_mf['buy_lg_amount'] + df_mf['buy_elg_amount']) - 
    (df_mf['sell_lg_amount'] + df_mf['sell_elg_amount'])
)

# 筛选最近30天的数据
df_recent = df_mf[df_mf['trade_date'] >= '20240101']

# 计算散户情绪指标（散户净流入占比）
df_recent['retail_sentiment'] = df_recent['retail_net_inflow'] / (
    df_recent['buy_sm_amount'] + df_recent['sell_sm_amount'] + 1e-6
) * 100

# 批量读取创业板股票资金流向
from data_utils import get_chinext_stocks
chinext_stocks = get_chinext_stocks()
moneyflow_data = {}
for code in chinext_stocks[:10]:  # 前10只
    file_path = f'{DATA_CENTER}/stock/moneyflow/{code}.parquet'
    if os.path.exists(file_path):
        moneyflow_data[code] = pd.read_parquet(file_path)
```

| 字段名              | 数据类型   | 说明     | 单位       | 用途          |
| ---------------- | ------ | ------ | -------- | ----------- |

### 9. Fama-French三因子（含金融股）

**绝对路径**: `/home/zy/桌面/数据中心/quant_data_center/factors/fama_french_3/ff_3_factors_daily.parquet`  
**用途**: 包含金融股的Fama-French三因子数据，用于全市场资产定价模型  
**更新频率**: 按需更新（支持月度TTM调仓）  
**构建方法**: 月度TTM方法，2x3 (Size × B/M) 排序  
**数据量**: 3,034条记录（2013年2月 ~ 2025年10月）  
**数据范围**: 2013-02-01 ~ 2025-10-30

#### 字段说明

| 字段名          | 数据类型    | 说明                     | 单位      |
| ------------ | ------- | ---------------------- | ------- |
| `trade_date` | string  | 交易日期（YYYYMMDD格式）       | -       |
| `MKT_RF`     | float64 | 市场风险溢价 = 市场收益率 - 无风险利率 | % (百分比) |
| `SMB`        | float64 | 规模因子 (Small Minus Big) | % (百分比) |
| `HML`        | float64 | 价值因子 (High Minus Low)  | % (百分比) |

#### 构建逻辑

1. **股票池**: 包含金融股的全市场股票（上市满1年）
2. **分组方法**: 
   - Size分组：市值中位数分割（S/B）
   - Value分组：B/M 30%/70%分位数（L/M/H）
   - 构建6个组合：SL, SM, SH, BL, BM, BH
3. **因子计算**:
   - `SMB = (R_SL + R_SM + R_SH)/3 - (R_BL + R_BM + R_BH)/3`
   - `HML = (R_SH + R_BH)/2 - (R_SL + R_BL)/2`
   - `MKT_RF = MKT - RF`（市值加权市场收益减去无风险利率）
4. **调仓频率**: 每月月末调仓，使用当月最后一天已公告的财务数据

#### 关键特性

- **包含金融股**: 与FF5因子不同，FF3因子包含金融股，适合全市场研究
- **月度TTM**: 使用Trailing Twelve Months财务数据，避免未来函数
- **市值加权**: 组合收益使用前一交易日市值加权计算
- **数据质量**: 无缺失值，数据完整性100%

#### 构建脚本

- **脚本路径**: `build_ff3_factors_full_market.py`
- **开始日期**: 2013-01-01（因子计算从2013年2月开始）
- **数据准备期**: 2012-01-01 ~ 2013-01-31（用于TTM计算）

#### 使用示例

```python

DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'

# 读取三因子数据（含金融股）

# 查看数据基本信息
print(f"数据量: {len(df_ff3)} 条")
print(f"日期范围: {df_ff3['trade_date'].min()} - {df_ff3['trade_date'].max()}")

df_stock['trade_date'] = pd.to_datetime(df_stock['trade_date'], format='%Y%m%d').dt.strftime('%Y%m%d')

# Fama-French三因子回归

X = df_merged[['MKT_RF', 'SMB', 'HML']].dropna()
y = df_merged.loc[X.index, 'return']

model = LinearRegression()
model.fit(X, y)

print("FF3因子回归结果:")
print(f"  Alpha: {model.intercept_:.4f}%")
print(f"  Beta_MKT: {model.coef_[0]:.4f}")
print(f"  Beta_SMB: {model.coef_[1]:.4f}")
print(f"  Beta_HML: {model.coef_[2]:.4f}")
print(f"  R²: {model.score(X, y):.4f}")
```

### 10. Fama-French五因子

**绝对路径**: `/home/zy/桌面/数据中心/quant_data_center/factors/fama_french_5/ff_5_factors_daily.parquet`  
**用途**: 用于因子回归、风险归因、残差动量、多因子模型等  
**更新频率**: 月度调仓（使用TTM财务数据）  
**数据量**: 3,096条记录（2012-11-01至2025-10-30）  
**构建脚本**: `build_ff5_factors_monthly_ttm.py`

#### 字段说明

| 字段名          | 数据类型   | 说明          | 单位       | 用途              |
| ------------ | ------ | ----------- | -------- | --------------- |
| `trade_date` | string | 交易日期        | YYYYMMDD | -               |
| `MKT_RF`     | float  | 市场风险溢价      | %        | 市场因子            |
| `SMB`        | float  | 规模因子（小减大）   | %        | Size效应          |
| `HML`        | float  | 价值因子（高减低）   | %        | Value效应         |
| `RMW`        | float  | 盈利因子（稳健减弱）  | %        | Profitability效应 |
| `CMA`        | float  | 投资因子（保守减激进） | %        | Investment效应    |

**因子说明**:

- **MKT_RF**: 市场组合收益率 - 无风险利率
- **SMB** (Small Minus Big): 小市值组合 - 大市值组合收益率
- **HML** (High Minus Low): 高账面市值比 - 低账面市值比收益率
- **RMW** (Robust Minus Weak): 高盈利能力 - 低盈利能力收益率
- **CMA** (Conservative Minus Aggressive): 低投资 - 高投资收益率

**TTM方法优势**:

- 使用滚动12个月（Trailing Twelve Months）财务数据，避免季节性影响
- 月度调仓，及时捕捉市场变化
- 严格遵守公告日期，避免未来函数

#### 使用示例

```python
import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据中心路径
DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'

# 读取五因子数据
df_ff5 = pd.read_parquet(f'{DATA_CENTER}/factors/fama_french_5/ff_5_factors_daily.parquet')

# 读取股票收益率数据
df_stock = pd.read_parquet(f'{DATA_CENTER}/stock/daily_hfq/000001.SZ.parquet')
df_stock['return'] = df_stock['close'].pct_change() * 100  # 转为%

# 合并数据
df_merged = df_stock.merge(df_ff5, on='trade_date', how='inner')

# Fama-French五因子回归
X = df_merged[['MKT_RF', 'SMB', 'HML', 'RMW', 'CMA']].values
y = df_merged['return'].values - df_merged['MKT_RF'].values  # 超额收益

model = LinearRegression()
model.fit(X, y)

alpha = model.intercept_  # Alpha值
beta_mkt, beta_smb, beta_hml, beta_rmw, beta_cma = model.coef_  # 因子暴露

print(f"Alpha: {alpha:.4f}%")
print(f"Beta (MKT): {beta_mkt:.4f}")
print(f"Beta (SMB): {beta_smb:.4f}")
print(f"Beta (HML): {beta_hml:.4f}")
print(f"Beta (RMW): {beta_rmw:.4f}")
print(f"Beta (CMA): {beta_cma:.4f}")

# 计算残差动量
df_merged['residual'] = y - model.predict(X)

# 风险归因
risk_contribution = {
    'MKT': beta_mkt * df_ff5['MKT_RF'].std(),
    'SMB': beta_smb * df_ff5['SMB'].std(),
    'HML': beta_hml * df_ff5['HML'].std(),
    'RMW': beta_rmw * df_ff5['RMW'].std(),
    'CMA': beta_cma * df_ff5['CMA'].std()
}
```

**用途**: 动量因子，用于衡量股票12-1月动量效应，作为v6.0回归模型的"噪音标尺"  
**更新频率**: 按需更新（支持月度调仓）  
**构建方法**: 月度调仓，2×3 (Size × MOM) 排序

#### 构建逻辑

1. **股票池**: 包含金融股的全市场股票（上市满1年）
2. **分组方法**: 
   - Size分组：市值中位数分割（S/B）

   - 高动量组合收益 - 低动量组合收益
4. **调仓频率**: 每月月末调仓，使用当月最后一天的市值和动量数据

#### 关键特性

- **包含金融股**: 全市场股票池
- **月度调仓**: 及时捕捉市场变化
- **市值加权**: 组合收益使用前一交易日市值加权计算
- **数据质量**: 无缺失值，数据完整性100%

| `trade_date` | string  | 交易日期（YYYYMMDD格式）              | -       |
| `LIQ`        | float64 | 流动性因子 (Illiquid Minus Liquid) | % (百分比) |

#### 构建逻辑

1. **股票池**: 包含金融股的全市场股票（上市满1年）
2. **分组方法**: 
   - Size分组：市值中位数分割（S/B）
   - LIQ分组：21日滚动平均换手率 30%/70%分位数（I/M/L）
   - LIQ计算：使用21日滚动平均换手率作为流动性代理
   - 低换手率 = Illiquid (I)，高换手率 = Liquid (L)
   - 构建6个组合：SI, SM, SL, BI, BM, BL
3. **因子计算**:
   - `LIQ = (R_SI + R_BI)/2 - (R_SL + R_BL)/2`
   - 低换手率组合收益 - 高换手率组合收益（流动性溢价）
4. **调仓频率**: 每月月末调仓，使用当月最后一天的市值和流动性数据

#### 关键特性

- **包含金融股**: 全市场股票池
- **月度调仓**: 及时捕捉市场变化
- **市值加权**: 组合收益使用前一交易日市值加权计算
- **流动性代理**: 使用21日滚动平均换手率，平滑短期波动
- **数据质量**: 无缺失值，数据完整性100%

### 11.2 中国版三因子 (CH-3)

**绝对路径**: `/home/zy/桌面/数据中心/quant_data_center/factors/ch_3_factors/ch_3_factors_daily.parquet`  
**用途**: 符合A股特性的"中国版三因子"，用于因子回归、风险归因等  
**更新频率**: 月度调仓（使用当日市值和PE数据）  
**构建方法**: 月度调仓，2×3 (Size × E/P) 排序，剔除殼價值  
**数据量**: 3,034条记录（2013年2月 ~ 2025年10月）  
**数据范围**: 2013-02-01 ~ 2025-10-30  
**构建脚本**: `build_ch3_factors.py`

#### 字段说明

| 字段名          | 数据类型    | 说明                     | 单位      |
| ------------ | ------- | ---------------------- | ------- |
| `trade_date` | string  | 交易日期（YYYYMMDD格式）       | -       |
| `MKT_RF`     | float64 | 市场风险溢价 = 市场收益率 - 无风险利率 | % (百分比) |
| `SMB_CH`     | float64 | 规模因子（小减大，中国版）         | % (百分比) |
| `VMG_CH`     | float64 | 价值因子（价值减成长，中国版）       | % (百分比) |

#### 构建逻辑

1. **股票池**: 包含金融股的全市场股票（上市满1年）
2. **剔除殼價值** (关键特性):
   - 每月排序时，首先按市值排序，剔除市值最小的30%股票
   - 保留剩余的70%股票进行因子构建
3. **分组方法**: 
   - Size分组：市值中位数分割（S/B）
   - Value分组：使用盈利收益率 (E/P = 1/PE) 替代账面市值比 (B/M)
   - E/P分组：30%/70%分位数（L/M/H）
   - 构建6个组合：SL, SM, SH, BL, BM, BH
4. **因子计算**:
   - `SMB_CH = (R_SL + R_SM + R_SH)/3 - (R_BL + R_BM + R_BH)/3`
   - `VMG_CH = (R_SH + R_BH)/2 - (R_SL + R_BL)/2`
   - `MKT_RF = MKT - RF`
5. **调仓频率**: 每月月末调仓，使用当月最后一天的市值和PE数据

#### 关键特性

- **包含金融股**: 全市场股票池
- **剔除殼價值**: 剔除市值最小的30%股票，避免殼价值影响
- **使用E/P**: 使用盈利收益率 (E/P) 代替账面市值比 (B/M)，更适合A股市场
- **月度调仓**: 及时捕捉市场变化
- **市值加权**: 组合收益使用前一交易日市值加权计算
- **数据质量**: 无缺失值，数据完整性100%

#### 与FF3因子的区别

| 特性   | CH-3因子（中国版）              | FF3因子（标准版）              |
| ---- | ----------------------- | ----------------------- |
| 股票池  | 包含金融股，剔除市值最小30%          | 包含金融股，不过滤               |
| 价值因子  | E/P (盈利收益率)             | B/M (账面市值比)             |
| 因子名称 | SMB_CH, VMG_CH         | SMB, HML                |
| 适用场景 | A股市场资产定价                 | 通用资产定价模型                |
| 因子构建 | 2×3 (Size × E/P)       | 2×3 (Size × B/M)        |

### 3. 计算残差动量

```python
import pandas as pd
from sklearn.linear_model import LinearRegression

def calculate_residual_momentum(ts_code, lookback_period=20):
    """计算残差动量因子"""
    # 数据中心路径
    DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'

    # 读取股票数据
    df_stock = pd.read_parquet(f'{DATA_CENTER}/stock/daily_hfq/{ts_code}.parquet')
    df_stock['return'] = df_stock['close'].pct_change()

    # 读取Fama-French五因子
    df_ff5 = pd.read_parquet(f'{DATA_CENTER}/factors/fama_french_5/ff_5_factors_daily.parquet')

    # 合并数据
    df = df_stock.merge(df_ff5, on='trade_date', how='inner')

    # 计算超额收益 (需要将收益率转为%)
    df['return_pct'] = df['return'] * 100
    df['excess_return'] = df['return_pct'] - df['MKT_RF']

    # 滚动回归计算残差
    residuals = []
    for i in range(lookback_period, len(df)):
        window_data = df.iloc[i-lookback_period:i]

        X = window_data[['MKT_RF', 'SMB', 'HML', 'RMW', 'CMA']].values
        y = window_data['excess_return'].values

        model = LinearRegression()
        model.fit(X, y)

        residual = y[-1] - model.predict(X[-1:])
        residuals.append(residual[0])

    df['residual_momentum'] = [np.nan] * lookback_period + residuals

    return df[['trade_date', 'residual_momentum']]
```

### 4. 多因子合成

```python
import pandas as pd
import numpy as np

def composite_factor(date, stock_pool):
    """多因子合成"""
    # 数据中心路径
    DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'

    factors = {}

    for code in stock_pool:
        # 读取各类因子
        df_stock = pd.read_parquet(f'{DATA_CENTER}/stock/daily_hfq/{code}.parquet')
        df_basic = pd.read_parquet(f'{DATA_CENTER}/stock/daily_basic/daily_basic_all.parquet')

        # 筛选日期
        stock_data = df_stock[df_stock['trade_date'] == date]
        basic_data = df_basic[(df_basic['ts_code'] == code) & (df_basic['trade_date'] == date)]

        if not stock_data.empty and not basic_data.empty:
            # 计算各因子
            momentum = stock_data['close'].pct_change(periods=20).iloc[-1]  # 动量
            value = 1 / basic_data['pb'].iloc[0]  # 价值（B/M）
            size = np.log(basic_data['total_mv'].iloc[0])  # 规模

            # 因子标准化
            factors[code] = {
                'momentum': momentum,
                'value': value,
                'size': size
            }

    # 转换为DataFrame
    df_factors = pd.DataFrame(factors).T

    # 标准化
    df_factors_normalized = (df_factors - df_factors.mean()) / df_factors.std()

    # 等权合成
    df_factors_normalized['composite'] = df_factors_normalized.mean(axis=1)

    return df_factors_normalized
```

## 📝 注意事项

### 1. 日期格式

所有日期字段统一使用 **YYYYMMDD** 格式（字符串类型）：

- ✅ 正确: `'20240101'`
- ❌ 错误: `'2024-01-01'`, `20240101`

### 2. 避免未来函数

使用财务数据时，必须使用 `ann_date`（公告日期）而不是 `end_date`（报告期），避免使用未来信息：

```python
# ❌ 错误：使用end_date会造成未来函数
df_fina = df_fina[df_fina['end_date'] == '20240331']

# ✅ 正确：使用ann_date避免未来函数
df_fina = df_fina[df_fina['ann_date'] <= '20240430']
```

### 3. 数据对齐

不同数据表的日期可能不一致（交易日 vs 财务报告期），需要注意数据对齐：

```python
# 使用merge时指定how='inner'确保日期对齐
df_merged = df_stock.merge(df_basic, on=['ts_code', 'trade_date'], how='inner')
```

### 4. 缺失值处理

部分股票或日期可能存在缺失值，需要妥善处理：

```python
# 检查缺失值
df.isna().sum()

# 前向填充
df.fillna(method='ffill', inplace=True)

# 删除缺失值
df.dropna(inplace=True)
```

### 5. 性能优化

处理大规模数据时，注意性能优化：

```python
# 1. 只读取需要的列
df = pd.read_parquet('file.parquet', columns=['ts_code', 'trade_date', 'close'])

# 2. 使用filter筛选数据
df = pd.read_parquet('file.parquet', filters=[('trade_date', '>=', '20240101')])

# 3. 批量处理避免逐行操作
# ❌ 慢
for i, row in df.iterrows():
    process(row)

# ✅ 快
df.apply(process, axis=1)
```

### 17. 前复权数据转换 🆕

**说明**: 数据中心同时存储后复权数据（daily_hfq）和前复权数据（daily_qfq）。前复权数据会在更新后复权数据时自动转换并保存到 `stock/daily_qfq/` 目录。

**自动转换**: 通过 `download_data_manager.py` 的 `update_stock_daily_hfq()` 方法自动生成  
**手动转换函数**: `data_utils.convert_hfq_to_qfq()`（用于实时转换或批量转换）

#### 使用示例

```python
from data_utils import convert_hfq_to_qfq
import pandas as pd

# 数据中心路径
DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'

# 方法1: 使用转换函数（推荐）
df_qfq = convert_hfq_to_qfq('000001.SZ', data_center_path=DATA_CENTER)
if df_qfq is not None:
    print(f"前复权数据: {len(df_qfq)} 条记录")
    print(df_qfq.head())

# 方法2: 批量转换
stock_codes = ['000001.SZ', '000002.SZ', '600519.SH']
qfq_data = {}
for code in stock_codes:
    qfq_data[code] = convert_hfq_to_qfq(code, data_center_path=DATA_CENTER)
```

**转换原理**:
- 前复权：以最新价格为基准，向前调整历史价格
- 后复权：以最早价格为基准，向后调整历史价格
- 转换公式：`qfq_price = hfq_price * (latest_price / hfq_price_at_latest_date)`

**注意事项**:
- 前复权数据已自动保存到 `stock/daily_qfq/` 目录，可直接读取使用
- 转换后的数据与后复权数据在最新日期价格一致
- 历史价格会按比例调整
- 如需实时转换（不保存文件），可使用 `data_utils.convert_hfq_to_qfq()` 函数
