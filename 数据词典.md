### 8. Fama-French五因子

**绝对路径**: `/home/zy/桌面/数据中心/quant_data_center/factors/fama_french_5/ff_5_factors_daily.parquet`  
**用途**: 用于因子回归、风险归因、残差动量、多因子模型等  
**更新频率**: 月度调仓（使用TTM财务数据）  
**数据量**: 3,094条记录（2012-11-01至2025-10-28）  
**构建脚本**: `build_ff5_factors_monthly_ttm.py`

#### 字段说明

| 字段名 | 数据类型 | 说明 | 单位 | 用途 |
|--------|---------|------|------|------|
| `trade_date` | string | 交易日期 | YYYYMMDD | - |
| `MKT_RF` | float | 市场风险溢价 | % | 市场因子 |
| `SMB` | float | 规模因子（小减大） | % | Size效应 |
| `HML` | float | 价值因子（高减低） | % | Value效应 |
| `RMW` | float | 盈利因子（稳健减弱） | % | Profitability效应 |
| `CMA` | float | 投资因子（保守减激进） | % | Investment效应 |

**因子说明**:
- **MKT_RF**: 市场组合收益率 - 无风险利率
- **SMB** (Small Minus Big): 小市值组合 - 大市值组合收益率
- **HML** (High Minus Low): 高账面市值比 - 低账面市值比收益率
- **RMW** (Robust Minus Weak): 高盈利能力 - 低盈利能力收益率
- **CMA** (Conservative Minus Aggressive): 低投资 - 高投资收益率

**TTM方法优势**:
- 使用滚动12个月（Trailing Twelve Months）财务数据，避免季节性影响
- 月度调仓，及时捕捉市场变化
- 严格遵守公告日期，避免未来函数

#### 使用示例

```python
import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression

# 数据中心路径
DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'

# 读取五因子数据
df_ff5 = pd.read_parquet(f'{DATA_CENTER}/factors/fama_french_5/ff_5_factors_daily.parquet')

# 读取股票收益率数据
df_stock = pd.read_parquet(f'{DATA_CENTER}/stock/daily_hfq/000001.SZ.parquet')
df_stock['return'] = df_stock['close'].pct_change() * 100  # 转为%

# 合并数据
df_merged = df_stock.merge(df_ff5, on='trade_date', how='inner')

# Fama-French五因子回归
X = df_merged[['MKT_RF', 'SMB', 'HML', 'RMW', 'CMA']].values
y = df_merged['return'].values - df_merged['MKT_RF'].values  # 超额收益

model = LinearRegression()
model.fit(X, y)

alpha = model.intercept_  # Alpha值
beta_mkt, beta_smb, beta_hml, beta_rmw, beta_cma = model.coef_  # 因子暴露

print(f"Alpha: {alpha:.4f}%")
print(f"Beta (MKT): {beta_mkt:.4f}")
print(f"Beta (SMB): {beta_smb:.4f}")
print(f"Beta (HML): {beta_hml:.4f}")
print(f"Beta (RMW): {beta_rmw:.4f}")
print(f"Beta (CMA): {beta_cma:.4f}")

# 计算残差动量
df_merged['residual'] = y - model.predict(X)
```

### 3. 计算残差动量

```python
import pandas as pd
from sklearn.linear_model import LinearRegression

def calculate_residual_momentum(ts_code, lookback_period=20):
    """计算残差动量因子"""
    # 数据中心路径
    DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'
    
    # 读取股票数据
    df_stock = pd.read_parquet(f'{DATA_CENTER}/stock/daily_hfq/{ts_code}.parquet')
    df_stock['return'] = df_stock['close'].pct_change()
    
    # 读取Fama-French五因子
    df_ff5 = pd.read_parquet(f'{DATA_CENTER}/factors/fama_french_5/ff_5_factors_daily.parquet')
    
    # 合并数据
    df = df_stock.merge(df_ff5, on='trade_date', how='inner')
    
    # 计算超额收益 (需要将收益率转为%)
    df['return_pct'] = df['return'] * 100
    df['excess_return'] = df['return_pct'] - df['MKT_RF']
    
    # 滚动回归计算残差
    residuals = []
    for i in range(lookback_period, len(df)):
        window_data = df.iloc[i-lookback_period:i]
        
        X = window_data[['MKT_RF', 'SMB', 'HML', 'RMW', 'CMA']].values
        y = window_data['excess_return'].values
        
        model = LinearRegression()
        model.fit(X, y)
        
        residual = y[-1] - model.predict(X[-1:])
        residuals.append(residual[0])
    
    df['residual_momentum'] = [np.nan] * lookback_period + residuals
    
    return df[['trade_date', 'residual_momentum']]
```

### 4. 多因子合成

```python
import pandas as pd
import numpy as np

def composite_factor(date, stock_pool):
    """多因子合成"""
    # 数据中心路径
    DATA_CENTER = '/home/zy/桌面/数据中心/quant_data_center'
    
    factors = {}
    
    for code in stock_pool:
        # 读取各类因子
        df_stock = pd.read_parquet(f'{DATA_CENTER}/stock/daily_hfq/{code}.parquet')
        df_basic = pd.read_parquet(f'{DATA_CENTER}/stock/daily_basic/daily_basic_all.parquet')
        
        # 筛选日期
        stock_data = df_stock[df_stock['trade_date'] == date]
        basic_data = df_basic[(df_basic['ts_code'] == code) & (df_basic['trade_date'] == date)]
        
        if not stock_data.empty and not basic_data.empty:
            # 计算各因子
            momentum = stock_data['close'].pct_change(periods=20).iloc[-1]  # 动量
            value = 1 / basic_data['pb'].iloc[0]  # 价值（B/M）
            size = np.log(basic_data['total_mv'].iloc[0])  # 规模
            
            # 因子标准化
            factors[code] = {
                'momentum': momentum,
                'value': value,
                'size': size
            }
    
    # 转换为DataFrame
    df_factors = pd.DataFrame(factors).T
    
    # 标准化
    df_factors_normalized = (df_factors - df_factors.mean()) / df_factors.std()
    
    # 等权合成
    df_factors_normalized['composite'] = df_factors_normalized.mean(axis=1)
    
    return df_factors_normalized
```

## 📝 注意事项

### 1. 日期格式

所有日期字段统一使用 **YYYYMMDD** 格式（字符串类型）：
- ✅ 正确: `'20240101'`
- ❌ 错误: `'2024-01-01'`, `20240101`

### 2. 避免未来函数

使用财务数据时，必须使用 `ann_date`（公告日期）而不是 `end_date`（报告期），避免使用未来信息：

```python
# ❌ 错误：使用end_date会造成未来函数
df_fina = df_fina[df_fina['end_date'] == '20240331']

# ✅ 正确：使用ann_date避免未来函数
df_fina = df_fina[df_fina['ann_date'] <= '20240430']
```

### 3. 数据对齐

不同数据表的日期可能不一致（交易日 vs 财务报告期），需要注意数据对齐：

```python
# 使用merge时指定how='inner'确保日期对齐
df_merged = df_stock.merge(df_basic, on=['ts_code', 'trade_date'], how='inner')
```

### 4. 缺失值处理

部分股票或日期可能存在缺失值，需要妥善处理：

```python
# 检查缺失值
df.isna().sum()

# 前向填充
df.fillna(method='ffill', inplace=True)

# 删除缺失值
df.dropna(inplace=True)
```

### 5. 性能优化

处理大规模数据时，注意性能优化：

```python
# 1. 只读取需要的列
df = pd.read_parquet('file.parquet', columns=['ts_code', 'trade_date', 'close'])

# 2. 使用filter筛选数据
df = pd.read_parquet('file.parquet', filters=[('trade_date', '>=', '20240101')])

# 3. 批量处理避免逐行操作
# ❌ 慢
for i, row in df.iterrows():
    process(row)

# ✅ 快
df.apply(process, axis=1)
```
